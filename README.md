# Bandit-kun

Monitors all reddit submissions and scrapes data from valid video submissions

## Dataset

Bandit-kun collects the following information from submissions

| **Header** | **Datatype** |
| :--------: | :----------: |
|   epoch    |   datetime   |
|   author   |     str      |
| subreddit  |     str      |
|   title    |     str      |
|   domain   |     str      |
|    url     |     url      |
|   width    |     int      |
|   height   |     int      |
|  duration  |     int      |
|  upvotes   |     int      |
|   ratio    |    float     |
|   score    |     int      |
|  guilded   |     int      |
| permalink  |    string    |
|    uuid    |     str      |
